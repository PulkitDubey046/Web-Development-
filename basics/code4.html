Ordered List of HTML Tags

  1. <html> – This tag wraps all the content on the entire web page as the root element.
  2. <head> – This tag holds meta-information about the document such as the title and links to CSS or JS files.
  3. <title> – This tag sets the web page title shown in the browser tab.
  4. <body> – This tag holds all the visible content of the web page.
  5. <h1> to <h6> – These header tags create headings, with <h1> being the largest and <h6> the smallest.
  6. <p> – This tag creates a text paragraph.
  7. <a> – This tag makes a hyperlink to another page or website.
  8. <img> – This tag puts images into the web page.
  9. <ul> – This tag makes an unordered (bulleted) list.
  10. <ol> – This tag makes an ordered (numbered) list.
  11. <li> – This tag represents a list item in ordered or unordered lists.
  12. <div> – This block-level container groups elements for styling or layout.
  13. <span> – This inline container styles a part of text or groups inline elements.
