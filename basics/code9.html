Features of HTML5 :-

1. Semantic Tags

HTML5 introduces meaningful tags like <section> and <article> for better structure and accessibility.

2. Multimedia Support

You can now add video and audio using <video> and <audio> without plugins.

3. New Form Inputs

HTML5 supports new input types like email, date, and range that improve user experience.
